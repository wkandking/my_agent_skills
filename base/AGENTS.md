# base

Shared knowledge used across roles.

## Read Base First When

- the task involves any write operation or worktree workflow
- the task involves credentials, secrets, or tokens
- the task involves long-running tasks or cross-session task state
- the task involves knowledge sedimentation
- the task involves skill creation
- the task involves shared repo workflow rules
