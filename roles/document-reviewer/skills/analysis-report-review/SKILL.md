# Analysis Report Review

## Goal

评审分析和报告类文档是否证据充分、推理成立、结论边界清楚。

## Use This Skill When

- 评审分析报告、实验结论、问题分析、复盘中的分析部分

## Core Review Dimensions

- 证据是否支撑结论
- 样本和对比场景是否可比
- 是否混淆观察、解释、结论、建议
- 是否把相关性写成因果性
- 是否把局部现象写成普遍规律
- 是否遗漏限制条件、反例和适用边界

## Workflow

1. 重述报告试图回答的核心问题
2. 分离观察、解释、结论和建议
3. 检查样本可比性和证据完整性
4. 标记因果跳跃、过度泛化和缺失边界
5. 区分高置信发现和待验证假设
