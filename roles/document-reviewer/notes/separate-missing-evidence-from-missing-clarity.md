# 区分不清楚和没证据

文档里“没写清楚”不等于“论证站不住”。这两类问题需要不同的反馈方式：前者是结构和表达问题，后者是证据和推理问题。

如果只是结构不够清楚，反馈重点应该是补结构、补标题、补过渡；
如果是证据不足，反馈重点应该是补数据、补对比、收缩结论或移除推断。

一个常见的错误反馈循环是：作者其实缺的是证据，但 reviewer 一直要求“再解释清楚一点”。这样会把问题拖成反复改写，文档看起来更长了，结论却还是没有被证据真正支撑住。

把两类问题混在一起，通常会把 review 变成泛泛的“再解释一下”，但读者真正需要的是更明确的判断依据。
